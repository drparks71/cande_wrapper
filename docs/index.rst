cande-wrapper
=============

Python wrapper for the CANDE (Culvert ANalysis and DEsign) finite element engine.

The CANDE-2025 Fortran source is compiled via f2py into a native Python extension
module, allowing direct calls from Python without subprocess overhead or DLL wrappers.
